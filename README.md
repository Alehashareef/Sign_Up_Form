    # Signup & Sign-in Form

    This is a simple web-based signup and sign-in form built with HTML, CSS, and JavaScript. 
    It allows users to register with their name, email, and password, and sign in using their credentials. 
    The user data is stored locally in the browser using localStorage.

    ## Features
    - **Sign Up**: Allows users to register with their name, email, and password.
    - **Sign In**: Allows users to log in using their email and password.
    - **Password Toggle**: View or hide the password input.
    - **Local Storage**: User data is stored locally, ensuring persistence across sessions.

    ## Technologies Used
    - **HTML**: Structure of the webpage.
    - **CSS**: Styling of the form and layout.
    - **JavaScript**: Form validation, localStorage operations, and password toggle functionality.

    ## Installation
    You can run this project locally by following these steps:
    1. Clone the repository:
       ```bash
       git clone https://github.com/yourusername/signup-signin-form.git
       ```
    2. Navigate to the project folder:
       ```bash
       cd signup-signin-form
       ```
    3. Open the `index.html` file in your browser to view the form.

    ## Usage
    - **Sign Up**: Enter a name, email, and password, then click "Register" to create an account.
    - **Sign In**: Enter your email and password, then click "Login" to access the application.

    ## Local Storage
    This form uses the browser's `localStorage` to save the registered users' data. It stores user details (name, email, and password) and validates login credentials using the saved information.
